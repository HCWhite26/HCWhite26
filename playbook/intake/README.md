# Intake Module

This folder contains all documentation, templates, and protocols related to the Intake phase of the Talent Acquisition lifecycle.

## Files in this folder

### **intake-recap-template.md**
A standardized template used to document and validate role alignment after every intake meeting.  
This recap becomes the single source of truth for:
- sourcing strategy  
- candidate evaluation  
- interview calibration  
- offer discussions  
- recalibration conversations  
- cross‑recruiter coverage  

Additional intake-related modules will be added here as the Playbook expands.
